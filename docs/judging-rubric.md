# Judging Rubric Mapping

**Shipping Mindset is weighted 2×.** Scope ruthlessly to one working demo flow.

| Criterion | Weight | How OvenMind wins it |
|---|---|---|
| Impact & Usefulness | 1× | Concrete $-impact estimates; real public FGF context (Listeria recall, CFO Industry 4.0 quote, Pickering build-out); real failure modes from OXMaint / iFactory. |
| Technical Execution | 1× | Genuine multi-agent LangGraph (not one LLM); real fine-tuned YOLO; real hybrid RAG; real PDF; real Slack webhook. Audit trail proves it. |
| Demo Quality | 1× | 90-second core flow; single-click trigger; dark control-room dashboard; live agent reasoning; projected $ impact. |
| Novelty & Creativity | 1× | **Cross-modal correlation** — most teams do single-modal. Bakery domain is specific and memorable. |
| **Shipping Mindset** | **2×** | Submit by 2pm Thursday. Working demo Friday. README + video + deployed URL + roadmap slide. Honest about simulated vs. real. |

## Milestone gates
- **Mon EOD:** one agent calling one tool; dashboard deployed; CV pipeline runs locally; RAG corpus written.
- **Tue EOD:** Trigger Scenario → sensors move, CV detects, one agent comments. Skeleton is alive.
- **Wed EOD:** full flow end-to-end (trigger → sensor → CV → correlation → RAG → PDF → Slack). Ugly but works.
- **Thu 1pm:** submit. Then rehearse 10×.

## Rule
If anything is broken at 9am Thursday, **cut it, don't fix it.** The only thing that must work is the 90-second core flow.
