# INC-2025-3403 — Labeling System BRA-LAB-1100-01, PLC communication fault

- **Date:** 2025-02-14
- **Line / Equipment:** PKG-Line-B / Labeling System (Packaging)
- **Severity:** Medium
- **Product affected:** Sliced bread bags

## Symptoms
Intermittent label mis-feeds: defect count rose from ~6 to **142** per shift and
defect rate from 0.9 % to **3.6 %**. Drive motor current showed brief stalls.
Sensor #2 reported `Degraded — Sensor #2` for several hours before the controller
raised **E2006 — PLC communication intermittent**. Vibration and motor
temperature remained near baseline — this was a controls fault, not a mechanical
one.

## Root cause
A loose terminal on the PLC's EtherCAT slave for Sensor #2 caused dropped
heartbeat messages. The motion controller defaulted to a fallback timing
profile that mis-timed the label applicator, producing the mis-feeds.

## Corrective action
Reseat and torque all EtherCAT terminals at the slave (**ECT-SL-2**), replace
the affected sensor (`SEN-LAB-2`), and update the PLC firmware to
**v4.2.1** which adds heartbeat-loss alarming.

## Impact
- **Downtime:** 1.8 hours
- **Cost:** $4,100
