# cv/synthetic/

Generate "good → defective" image pairs with **Stable Diffusion + ControlNet**
(burnt / uneven browning naan, muffins, croissants). Technique proven for food
defect detection (Li et al., "A Candy Defect Detection Method Based on StyleGAN2
and Improved YOLOv7", MDPI Foods 13(20):3343, 2024).

Put generation scripts/prompts and the resulting images here, then label and fold
into `../datasets/`. Generated images are gitignored.
